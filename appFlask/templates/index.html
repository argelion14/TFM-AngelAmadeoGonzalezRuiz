    About the Project

    This web application is the result of an academic final project that implements a comprehensive permission management system based on DDS RTI Security XML policies. It enables the creation, assignment, validation, and visualization of users, roles, and security permissions through JWT tokens.

    Its purpose is to simulate real-world RBAC and permission validation processes for distributed systems using cryptographic certificates.

    Available Features

    • Users:
      • Create, list, view details, and delete users
      • Generate and manage public/private certificates
    • Roles:
      • Define new roles with expiration time
      • Associate roles to users
      • Assign roles based on grant templates
    • Grant Templates:
      • Upload and parse DDS Permissions XML files
      • View associated rules, domains, and topics
    • JWT Token Management:
      • Create tokens with cryptographic signature
      • Validate tokens and decode payload
    • Dashboard:
      • See summary statistics about users, roles, grants, etc.
